Massive Wiki Publishing

You can use many applications to view and edit Massive Wikis.

Markpub converts a Massive Wiki into a static HTML website, for easy viewing on the web. Markpub will recreate the main Massive Wiki website from its Markdown source files every time someone pushes changes to the associated GitHub repository.

You can see Markpub on the Massive Wiki Conceptual Diagram, in the center.
